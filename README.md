## ğŸ§  Engineering Highlights

### **Hybrid Auth Integration**
The dashboard uses a common `NEXTAUTH_SECRET` to verify tokens generated by the Next.js parent app. This allows a user to log in on the main site and access the dashboard without redundant logins, provided the domain-sharing or token-passing logic is met.

### **Server-State Management**
Instead of polluting global state (Redux/Context) with server data, we use **React Query**. This ensures that the data is always fresh, eliminates unnecessary API calls, and provides out-of-the-box loading/error states for a superior UX.
